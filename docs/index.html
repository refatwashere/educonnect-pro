Quick Start

1. Clone the Repository

git clone https://github.com/refatwashere/educonnect-pro.git
cd educonnect-pro

2. Start Development Servers

# Windows
scripts\start-dev.bat

# Linux/Mac
scripts/start-dev.sh

3. Access the Application

Frontend:
localhost:3000
Backend API:
localhost:8000
API Docs:
localhost:8000/docs
Default Login:
Username: teacher@school.edu
Password: password
